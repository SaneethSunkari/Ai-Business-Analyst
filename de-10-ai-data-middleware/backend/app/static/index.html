    Agent Tools — OpenAI Function-Calling Format

    Any AI agent (OpenAI, Claude, LangChain, etc.) can plug into this middleware. Fetch the manifest and paste it into your agent's tools array — your agent can then call these tools directly.

    test_connection
    Verify a data source connection. Use inline credentials or a saved connection_id.
    GET /tools/manifest → full OpenAI tools array POST /tools/invoke → execute any tool by name
    register_connection
    Save credentials once, get a connection_id the agent reuses on every call — no repeated credential passing.
    query_database
    Ask a plain-English question. The middleware handles schema lookup, SQL generation, and execution.
    POST /tools/invoke { "tool": "query_database", "arguments": { "connection_id": "<id>", "question": "How many patients are there?" } }
    run_sql
    Execute a raw read-only SELECT query.
    inspect_schema
    Return all tables, columns, and relationships so the agent understands the active source model.
    Fetch manifest with curl
    curl http://localhost:8000/tools/manifest
